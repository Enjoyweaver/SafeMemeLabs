#### How do you know what the risks are when creating a new token and how to mitigate them?

Most of the risks below can be mitigated with the SafeMeme token and SafeLaunch mechanisms, though we will elaborate
on how we address all of the risks further below.

When you want to deploy a token, how do you know which blockchain to deploy it on and how do you know
if you can you create a token that is compatible with the blockchain?

For example, if you chose to deploy a token on the Solana blockchain, you would have to learn the
progamming language Rust (unless you already knew it) and would have to learn the process of
deploying a token on Solana, which is not simple or easily done
